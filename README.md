# Algoritmi per Big Data

![GitHub release (latest by date)](https://img.shields.io/github/v/release/mirco0/AlgoritmiBigData?label=Ultima%20Versione)

Raccolta di appunti scritti in **LaTeX** per il corso di **Algoritmi per Big Data**. 
L'obiettivo di questa repository è mantenere una versione aggiornata, corretta e facilmente consultabile del materiale di studio.

La versione compilata del documento è disponibile per il download nella sezione **[Release](https://github.com/mirco0/AlgoritmiBigData/releases/latest)**

---

## Argomenti Trattati
1.  **Richiami di Probabilità**: Fondamenti di teoria, variabili aleatorie, distribuzioni (binomiale, geometrica) e teoremi limite, con Chernoff Bounds e problema del Coupon Collector.
2.  **Algoritmi Probabilistici**: Metodologie per la verifica di identità polinomiali, moltiplicazione matriciale, risoluzione del problema Min-Cut e analisi del QuickSort randomizzato.
3.  **Tecniche di Hashing**: Studio delle famiglie di funzioni hash universali, perfect hashing e implementazione di tabelle hash per la gestione efficiente del problema del dizionario.
4.  **Similarità tra Documenti**: Analisi delle tecniche di Shingling, Min-Hashing e Locality Sensitive Hashing (LSH) per l'individuazione di elementi simili in dataset ad alta dimensionalità.
5.  **Streaming di Dati**: Gestione di flussi di dati continui, tecniche di campionamento (Reservoir Sampling), algoritmi di sketching (Min-Sketch), Filtri di Bloom e calcolo dei momenti mediante il metodo AMS.
